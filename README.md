# gotchiLink 💻🌐

A modern, clean, and mobile-friendly web dashboard for Pwnagotchi. 
Designed to replace the legacy web UI with a high-performance, single-page interface.

## ✨ Features
- **Modern UI**: Clean dark theme inspired by Pwnagotchi Link.
- **Native Performance**: Uses raw Linux system calls for accurate CPU, RAM, and Temp data.
- **Handshake Management**: Download or delete .pcap files directly from the dashboard.
- **Real-time Stats**: Reordered stats (RAM -> CPU -> Temp) to match community standards.
- **PWND Badge**: Displays your lifetime handshake total at a glance.
